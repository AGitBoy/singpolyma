ABM Locator test page

Enter a Canadian city and province in the text box and click "Submit" to see a listing of the addresses and other meta-data returned by the web scraper code for ABM Locator. A map will soon be added that plots the addresses and meta-data. Note that the submission process may take up to 5 minutes, depending on the number of ABMs that are returned and how long ABM Locator has to wait for the banking sites to return results.
